Q1. WHY IS MY APP NOT RUNNING?
Ans. This is possible due to the fact you may not have had the KIVYMD and KIVY packages installed on startup.
To rectify this delete and reinstall the application or if you know how to use the pip command, Type

pip install KIVY
pip install KIVYMD
in the terminal
Q2. APP IS RUNNING BUT IT WONT LET ME LOGIN?
Ans. This error is occuring due to the fact you have probably not entered the correct mysql password or you may not have mysql server not installed at all.
This may happen if your password is SHA2 protected.
Please shift to legacy password to connect seamlessly if this has been your issue. Else simply delete full app and reinstall it
